# 🔐 Smart Password Generator

A Python-based tool that generates strong, secure, and customizable passwords using randomness and user-defined preferences.

---

## 🚀 Features
- Generate secure random passwords
- Customizable password length
- Option to include symbols for stronger security
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- Python
- Built-in libraries: `random`, `string`

---

```bash
# Clone the repository

# Navigate to project folder
